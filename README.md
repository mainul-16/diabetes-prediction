# 🩺 Diabetes Prediction System

A machine learning web application that predicts the likelihood of diabetes based
on patient medical data. Built with **Gradio** and deployed on **Streamlit Cloud**.

---

## 📁 Project Structure

```
diabetes-prediction/
│
├── data/
│   └── diabetes.csv              ← Download from Kaggle (see below)
│
├── model/
│   ├── diabetes_model.pkl        ← Generated by train_model.py
│   └── scaler.pkl                ← Generated by train_model.py
│
├── .streamlit/
│   └── config.toml               ← Streamlit Cloud theme & server config
│
├── app.py                        ← Gradio UI app
├── streamlit_app.py              ← Streamlit wrapper (entry point for cloud)
├── train_model.py                ← Train & save the model locally
├── requirements.txt              ← All Python dependencies
└── README.md
```

## 🚀 Step-by-Step: Local Setup

### Step 1 — Clone or create the project folder

```bash
mkdir diabetes-prediction
cd diabetes-prediction
```

### Step 2 — Create a virtual environment

```bash
python -m venv venv

# Activate it:
# On Windows:
venv\Scripts\activate
# On Mac/Linux:
source venv/bin/activate
```

### Step 3 — Install dependencies

```bash
pip install -r requirements.txt
```

### Step 4 — Download the dataset

1. Go to: https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database
2. Download `diabetes.csv`
3. Place it inside the `data/` folder:

```
diabetes-prediction/
└── data/
    └── diabetes.csv
```

### Step 5 — Train the model

```bash
python train_model.py
```

This will create two files:
- `model/diabetes_model.pkl`
- `model/scaler.pkl`

You should see output like:
```
✅ Accuracy : 0.7922
✅ ROC-AUC  : 0.8561
✅ model/diabetes_model.pkl  saved
✅ model/scaler.pkl          saved
```

### Step 6 — Run the app locally

```bash
python app.py
```

Then open your browser at: **http://localhost:7860**

## ☁️ Step-by-Step: Deploy to Streamlit Cloud

### Step 1 — Push your project to GitHub

Make sure your repo contains these files:
- `app.py`
- `streamlit_app.py`
- `requirements.txt`
- `.streamlit/config.toml`
- `model/diabetes_model.pkl`  ← **Must be committed to the repo**
- `model/scaler.pkl`          ← **Must be committed to the repo**

> ⚠️ IMPORTANT: The `.pkl` model files must be in your GitHub repo.
> Streamlit Cloud does NOT run `train_model.py` — it only installs packages
> and runs `streamlit_app.py`. You need to commit the trained model files.

```bash
git init
git add .
git commit -m "Initial commit — Diabetes Prediction App"
git branch -M main
git remote add origin https://github.com/mainul-16/diabetes-prediction.git
git push -u origin main
```

### Step 2 — Sign in to Streamlit Cloud

1. Go to: https://streamlit.io/cloud
2. Click **"Sign in with GitHub"**
3. Authorize Streamlit to access your GitHub account

### Step 3 — Deploy your app

1. Click **"New app"**
2. Fill in the form:
   - **Repository:**  `YOUR_USERNAME/diabetes-prediction`
   - **Branch:**      `main`
   - **Main file path:** `streamlit_app.py`
3. Click **"Deploy!"**

Streamlit Cloud will:
- Install all packages from `requirements.txt`
- Start the app (takes ~2–3 minutes on first deploy)
- Give you a public URL like:
  `https://your-username-diabetes-prediction-streamlit-app-xxxx.streamlit.app`

### Step 4 — Share your app

Copy the public URL and share it with anyone — no installation needed on their end!

## 🔄 How the Architecture Works

```
User's Browser
      │
      ▼
Streamlit Cloud  (runs streamlit_app.py on port 8501)
      │
      │  launches in background thread
      ▼
Gradio Server    (runs app.py on port 7860)
      │
      │  embedded via <iframe>
      ▼
User sees the Gradio UI inside the Streamlit page
```

## 🧠 Model Details

| Property       | Value                              |
|----------------|------------------------------------|
| Algorithm      | Random Forest Classifier           |
| Dataset        | PIMA Indian Diabetes Dataset       |
| Training split | 80% train / 20% test               |
| Accuracy       | ~79%                               |
| ROC-AUC        | ~0.86                              |
| Features used  | 8 medical measurements             |

### Features

| Feature                  | Description                        |
|--------------------------|------------------------------------|
| Pregnancies              | Number of pregnancies              |
| Glucose                  | Plasma glucose (mg/dL)             |
| BloodPressure            | Diastolic BP (mm Hg)               |
| SkinThickness            | Triceps skin fold (mm)             |
| Insulin                  | 2-Hr serum insulin (mu U/ml)       |
| BMI                      | Body mass index (kg/m²)            |
| DiabetesPedigreeFunction | Family history likelihood score    |
| Age                      | Patient age (years)                |
